# DLPS Network Viewer

An offline DB viewer to quickly search, browse, and find a specific link from the DLPS network - user has to provide their own cache. 

Open `dlpsNetworkViewer.html` in your browser to quickly inspect, search, filter and export cache contents without running a server.

---

## ✅ Features

- Load a cache bundle (`.zip`) or one/more `games_cache_*.json` files
- Live search across titles, URLs, sources and categories
- Filters for Source, Category, Link status and Link type
- Per-game modal to view all download links
- CSV export of the currently filtered results

---

## Quick start

1. Open the viewer:
   - Double-click `dlps_cache_viewer/dlpsNetworkViewer.html` 

2. Load data:
   - Click **Upload cache bundle (.zip) or a JSON file** and select a `dlpsNetworkCache.zip` or one of the `games_cache_*.json` files. That you have sourced elsewhere

- Supported inputs:
- `games_cache_*.json` — standard cache files generated by the cacher.
- `skipped.json` is detected and used only for skip-list collection.

---

## Notes

- Users must source/locate their own cache files. I will not provide them within this repo
- Large caches may take a few moments to parse
